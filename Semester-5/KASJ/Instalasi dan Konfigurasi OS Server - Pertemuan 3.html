Network Interface Configuration

Windows Server

# PowerShell Commands New-NetIPAddress -InterfaceIndex 12 ` -IPAddress "192.168.1.100" ` -PrefixLength 24 ` -DefaultGateway "192.168.1.1" Set-DnsClientServerAddress ` -InterfaceIndex 12 ` -ServerAddresses "8.8.8.8","8.8.4.4"
GUI Method:
Network and Sharing Center → Change adapter settings

Linux Server

# Ubuntu Netplan Configuration
# /etc/netplan/00-installer-config.yaml

network:
    version: 2
    ethernets:
        ens33:
            dhcp4: false
            addresses:
                - 192.168.1.100/24
            gateway4: 192.168.1.1
            nameservers:
                addresses: [8.8.8.8, 8.8.4.4]
sudo netplan apply sudo systemctl restart networking

Common Installation Issues
Troubleshooting & Solutions
Issue Symptoms Solution
Boot Failure System won't boot from installation media Check BIOS/UEFI settings, verify media integrity
Network Issues No network connectivity after installation Check cable, driver installation, IP configuration
Driver Problems Hardware not recognized Download latest drivers from manufacturer
License Issues Activation failures (Windows) Verify license key, check internet connectivity
Disk Space Installation fails due to insufficient space Free up space, use different partition
