üí° Defense in Depth Strategy

Implementasi berlapis-lapis dengan redundant security controls untuk memastikan bahwa kegagalan satu layer tidak mengkompromikan keseluruhan sistem. Setiap layer memiliki kontrol yang independen namun terintegrasi.

Detection Time: Reduce mean time to detection (MTTD) < 1 hour
Response Time: Achieve mean time to response (MTTR) < 4 hours
Recovery Time: Complete recovery objective < 24 hours

SIEM Rule Example (Splunk SPL):
index=security sourcetype=linux_audit
| search "failed password"
| stats count by src_ip user
| where count > 10
| eval risk_score = case(
  count > 50, "high",
  count > 20, "medium",
  1=1, "low")
| table src_ip user count risk_score
